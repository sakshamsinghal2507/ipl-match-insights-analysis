# IPL Match Insights & Performance Analysis

## 📌 Project Overview
This project analyzes IPL dataset to uncover insights about team performance, player statistics, and match trends.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Analysis Performed
- Team-wise match wins
- Top batsmen analysis
- Toss impact on match results
- Venue analysis
- Correlation heatmap

## 🔍 Key Insights
- Some teams win more matches consistently
- Toss has a partial impact on match outcomes
- Top players contribute significantly to total runs
- Certain venues host more matches than others

## 📁 Files Included
- ipl_analysis.ipynb
- matches.csv
- deliveries.csv

## 🚀 Conclusion
This project demonstrates basic to intermediate data analysis skills and visualization techniques using real-world sports data.
